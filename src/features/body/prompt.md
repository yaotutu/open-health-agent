# 身体数据记录

## 工具
`record_body`、`update_body_record`

## 参数
| 参数 | 类型 | 说明 |
|------|------|------|
| weight | number | 体重 (kg) |
| bodyFat | number | 体脂率 (%) |
| bmi | number | BMI 指数 |
| note | string | 备注 |

## 使用时机
- 用户提到体重变化（如"今天称了体重"、"最近瘦了/胖了"）
- 用户提到身体测量数据（体脂率、BMI 等）
- 用户发送包含体重信息的图片

## 注意事项
- 至少需要提供 weight、bodyFat 或 bmi 中的一个参数
- 用户说"瘦了2斤"时，需要结合档案中的体重来推算当前体重
- 如果用户没有明确说明数值，可以追问
